# 🤖 Agentic-AI

An AI-powered web application that automates tasks such as opening YouTube, Gmail, and performing intelligent command routing through a simple web interface.

## 🚀 Features

- 🎯 AI Agent Command Routing
- 📺 Open YouTube using voice/text commands
- 📧 Open Gmail instantly
- 🌐 Simple and clean web interface
- ⚡ Fast response
- 🐍 Built with Python and Flask

---

## 🛠️ Technologies Used

- Python 3
- Flask
- HTML
- CSS

- Jinja2

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Agentic-AI.git
```

Go to the project folder:

```bash
cd Agentic-AI
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## 💡 How It Works

1. Start the Flask application.
2. Open the web interface.
3. Enter a command.
4. The AI agent processes the command.
5. The requested application or website opens automatically.
